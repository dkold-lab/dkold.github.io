# EbanoE GovnO 2

**EbanoE GovnO 2** — учебный 3D FPS-прототип на Python + Panda3D: вид от первого лица, стрельба, боты, деньги, кейсы и карта, вдохновлённая классической схемой Dust 2 без копирования оригинальных ассетов.

## Возможности

- Вид от первого лица: WASD, мышь, прыжок, бег.
- Оружие: AK-подобная винтовка, M4-подобная винтовка, AWP-подобная снайперка, пистолет.
- Экономика: деньги за убийства, покупка кейсов, открытие скинов.
- Боты с простым ИИ: патрулируют, видят игрока, стреляют и получают урон.
- Карта `Sandy Crossing`: mid, long, B tunnels, A/B sites — оригинальная учебная геометрия в духе tactical FPS.
- Поддержка внешних weapon geometry-ассетов: скрипт загрузки кладёт модели в `assets/cs2_weapon_model_geometry/`, игра пытается использовать их как визуальные модели оружия, а при отсутствии ассетов включает процедурную модель.

> Важно: это фанатский/учебный прототип. Он не содержит карт, текстур, звуков или игровых файлов Counter-Strike 2. Используйте сторонние ассеты только в рамках их лицензии и правил Valve/Steam Workshop.

## Установка

```bash
cd ebanoe_govno2
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Загрузка weapon geometry моделей

```bash
python asset_downloader.py
```

Скрипт скачивает архив по официальной ссылке Steam CDN, распаковывает его в `assets/cs2_weapon_model_geometry/` и создаёт `assets/asset_manifest.json`. Если сеть недоступна или доступ запрещён, игра всё равно запускается с процедурным оружием.

## Запуск

```bash
python game.py
```

## Управление

| Клавиша / мышь | Действие |
| --- | --- |
| W A S D | Движение |
| Mouse | Обзор |
| ЛКМ | Стрелять |
| R | Перезарядка |
| Shift | Бег |
| Space | Прыжок |
| 1-4 | Сменить оружие |
| B | Купить кейс ($700) |
| O | Открыть кейс |
| Esc | Захват/освобождение мыши |
| F1 | Перезапуск раунда |

## Структура

- `game.py` — сама игра.
- `asset_downloader.py` — загрузка и распаковка моделей оружия.
- `requirements.txt` — зависимости.
